# Interview Scheduler

![](public/images/scheduler.gif)

## Setup

Install dependencies with `npm install`.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```

## Dependencies

1. Axios
2. Babel
3. Webpack
4. React-Hooks
5. Node-Sass
6. Prop-types
7. Jest
8. Cypress
